How to Run
1

Back Up Your Data First

Go to Facebook Settings → Your Facebook Information → Download Your Information. This is irreversible!

2

Open Activity Log

Log into Facebook → click your profile picture → click Activity Log. Make sure you're on the correct filter tab (Posts, Likes, etc.).

3

Open Browser Console

Press F12 (Windows/Linux) or Cmd+Option+J (Mac) to open DevTools. Click the Console tab.

4

Paste & Execute

Paste the generated script into the console and press Enter. The script will log its progress. Run it again after scrolling for more items.

5

Stop Anytime

Type window.__FB_ERASER_STOP = true in the console to abort the script mid-run.
